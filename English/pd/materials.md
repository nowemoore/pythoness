In this section, you'll learn about the possibilities of showing your code to the world, incl. possible collaborators or employers.

## üîß GitHub

1) **`readme` documentation:** A markdown file in your repository that explains what your project does, how to install it, and how to use it.
2) **in-line comments:** Brief explanations using # within your code that describe what specific lines or sections do.
3) **GitHub pages:** A free hosting service that turns your repository into a live website to showcase your project documentation or portfolio.

## üåê LinkedIn

1) **Skills**: The programming languages, tools, and technologies you list in your LinkedIn profile (like Python, Git, VS Code).
